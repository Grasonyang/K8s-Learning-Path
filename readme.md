# 專案目標

本專案的目標是使用現有的 Docker 容器來構建**多個節點**的 Kubernetes (k8s) 環境。透過此專案，您將能夠快速部署和管理容器化應用程式，並利用 Kubernetes 提供的強大功能來進行容器編排和管理。

- 使用現有的 Docker 容器
- 構建多個節點的 Kubernetes 環境
- 快速部署和管理容器化應用程式
- 利用 Kubernetes 進行容器編排和管理
